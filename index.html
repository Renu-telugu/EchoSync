How to Play EchoSync

1. The game will light up one button.

2. Click on that same button.

3. The game will add another button to the sequence.

4. Remember and click the entire sequence in order.

5. The game continues adding one button to the sequence each round.

6. If you make a mistake, the game ends and your score is displayed.
